# Ubiquitous Language

- `Community`: one external community, such as V2EX or 过早客.
- `ExternalAccount`: one handle on one community.
- `IdentityCluster`: an analysis subject composed of one or more external accounts.
- `CommunityProfileSnapshot`: a profile snapshot produced by a connector during one request.
- `CanonicalActivity`: a normalized `topic` or `reply` item with traceable origin.
- `Evidence`: an excerpt attached to a portrait claim.
- `Portrait`: the structured output of rule-based analysis.
- `CommunityBreakdown`: per-community portrait summary.
